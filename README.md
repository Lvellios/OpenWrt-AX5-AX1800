# Redmi AX5
- RAM: 256M
- Flash: 128M
- SoC: IPQ6000 (Quad Cores A53@1.0-1.2 Ghz, 1 NPU@1.5 GHz)
- Switch Chip: IPQ8075
- 5G Wireless Chip: QCN5052
- Front End Module: QPF4550 ×2
- 2.4 Wireless Chip: QCN5022
- Front End Module: QPF4200 ×2
